## 구현 기능
---
+ 계정 관련
+     로그인, 로그아웃, 회원가입, 회원탈퇴
+     관리자 페이지를 통한 영화 조회, 유저 관리, 코멘트/댓글 삭제 및 상세 검색 기능
+ 코멘트
+     코멘트 CRUD
+     댓글, 대댓글 CRUD
+     유저의 코멘트 내용 검색
+     최신순, 댓글순, 조회수 순 정렬
+ 평점
+     코멘트와 별개로 별점으로 평가 가능(0.5점~5.0점)
+ 영화 관련
+     키워드와 일치하는 영화 검색
+ 마이 페이지
+     내가 남긴 평점, 코멘트 리스트 확인
+     캘린더 기능
